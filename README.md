# ICA 01: Vote

This project implements a ranked-choice voting system in Python.
Each voter ranks candidates based on the absolute distance between their political leaning and each candidate's leaning. Voter vote for their highest-ranked candidate who has not been eliminated.

If no candidate receives a majority of >50% of votes, the candidate with the fewest votes is eliminated each round until a winner is determined.
